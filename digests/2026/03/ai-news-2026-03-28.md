## Claude Code / Anthropic

### Claude Code v2.1.86リリース（3/27）
多数のバグ修正とパフォーマンス改善。主要な変更点:
- APIリクエストに `X-Claude-Code-Session-Id` ヘッダー追加（プロキシでのセッション集約用）
- Jujutsu（`.jj`）/ Sapling（`.sl`）のVCSディレクトリを除外リストに追加
- **Opus 4.6がMax/Teamでデフォルト「medium」エフォートに変更**。高エフォートが必要な場合は「ultrathink」キーワードを使用
- **Opus 4 / 4.1をClaude Code（ファーストパーティAPI）から削除**。ピン留めユーザーはOpus 4.6に自動移行
- `--resume`、`--bare`モード、`/feedback` OOM、Windows設定破損、VSCode RTLテキスト描画など多数のバグ修正
- Readツールのトークン使用量削減（コンパクト行番号フォーマット）
- https://code.claude.com/docs/en/changelog
