# AI News Digest - 2026-04-01

## Claude Code

### Claude Code v2.1.88（2026-03-30）
前日チェック時（3/31）にはv2.1.87が最新だったが、v2.1.88がリリースされていた。41件の変更を含む大型アップデート。

**新機能:**
- `CLAUDE_CODE_NO_FLICKER=1` 環境変数 — ちらつきのないalt-screenレンダリング（仮想スクロールバック）
- `PermissionDenied` フック — auto modeの分類器拒否後に発火、`{retry: true}` でリトライ指示可能
- 名前付きサブエージェントが `@` メンションのタイプアヘッド候補に表示

**パフォーマンス改善:**
- Opus 4.6のデフォルト最大出力トークンを64kに引き上げ、上限を128kに拡大（Sonnet 4.6も同様）
- 多数のMCPコネクタ設定時の起動イベントループ遅延を軽減（macOS keychainキャッシュ 5s→30s）
- `@` ファイルメンション時のトークンオーバーヘッド削減（JSON escapeを廃止）
- 起動時間を約30ms改善（setup()の並列実行）

**主要バグ修正:**
- 長いセッションでのプロンプトキャッシュミス修正
- ネストした `CLAUDE.md` が長セッションで数十回再注入される問題を修正
- StructuredOutputスキーマキャッシュバグ修正（約50%の失敗率を解消）
- Edit/Writeツールのwindows CRLF二重化・Markdownハードラインブレーク除去を修正
- 公式マーケットプレイスプラグインの "Permission denied" エラー修正（v2.1.83以降の退行）

**その他:**
- issue/PR参照は `owner/repo#123` 形式のみクリック可能リンクに（bare `#123` は対象外）
- 現在の認証設定で使用不可のスラッシュコマンドを非表示化
- [VSCode] レート制限警告バナー追加（使用率・リセット時間表示）
- システムプロンプト変更: エージェントガイダンスに「理解の委任禁止」追加

Source: [Claude Code Changelog](https://code.claude.com/docs/en/changelog)
