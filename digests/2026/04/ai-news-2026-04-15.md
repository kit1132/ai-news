# AI News Digest - 2026-04-15

## Anthropic / Claude

### Claude Managed Agents パブリックベータ開始（4/8）
Anthropicが自律エージェント実行基盤「Claude Managed Agents」をパブリックベータで公開。セキュアサンドボックス、ビルトインツール（web search、code execution、bash、computer use等）、SSEストリーミングを備えた完全マネージドハーネス。APIトークン料金 + $0.08/session-hour。outcomes・multiagent・memoryはリサーチプレビュー。
- https://platform.claude.com/docs/en/managed-agents/overview
- https://claude.com/blog/claude-managed-agents

### ant CLI リリース（4/8）
Claude API用の公式CLIクライアント。Claude Codeとのネイティブ統合、YAMLによるAPIリソースのバージョニング対応。Managed Agentsのデプロイにも使用可能。
- https://github.com/anthropics/anthropic-cli

### Claude Code v2.1.101〜v2.1.107（4/10-14）
5日間で3リリース。主な変更点:

**v2.1.107（4/14）**: thinkingヒントの早期表示

**v2.1.105（4/13）**:
- `EnterWorktree` ツールに `path` パラメータ追加（既存worktreeへの切替）
- PreCompactフック: exit code 2 or `{"decision":"block"}` でコンパクション阻止
- バックグラウンドプラグインモニター（`monitors` マニフェストキー）
- `/proactive` → `/loop` のエイリアス化
- WebFetchが `<style>`/`<script>` 内容を除去（CSS重いページの改善）
- APIストリーム5分無データで自動アボート→非ストリーミングリトライ
- `/doctor` にステータスアイコン、`f` キーで修正実行

**v2.1.101（4/10）**:
- `/team-onboarding` コマンド（チーム向けオンボーディングガイド生成）
- OS CA証明書ストアをデフォルト信頼（企業TLSプロキシ対応）
- Vertex AI対話型セットアップウィザード
- コマンドインジェクション脆弱性修正（POSIX `which` フォールバック）
- 長時間セッションのメモリリーク修正
- Bedrock SigV4認証の403エラー修正
- 多数のresume/plugin/rendering/keybinding修正

- https://code.claude.com/docs/en/changelog

## OpenAI

### GPT-5.4 API リリース
Chat Completions APIとResponses APIでGPT-5.4（フロンティアモデル）およびGPT-5.4 pro（高計算量版）が利用可能に。GPT-5.3 Instant Miniも新フォールバックモデルとしてChatGPTに追加。
- https://platform.openai.com/docs/changelog

### GPT-5.5 "Spud" — リリース間近
事前学習完了（3/24）。Sam Altmanが「経済を大幅に加速しうる非常に強力なモデル」と社内説明。Polymarket: 4月中リリース78%。GPT-5.5かGPT-6かはベンチマーク次第（GPT-5.4比40%+向上ならGPT-6の可能性）。
- https://lumichats.com/blog/gpt-5-5-spud-openai-release-date-features-april-2026-complete-guide

### ChatGPT 新機能
- **ショッピング改善**: チャット内での商品閲覧・会話による絞り込み・横並び比較
- **Libraryファイル管理**: アップロード/生成ファイルの自動保存・検索・サイドバー参照
- **Outlook連携拡張**: 共有メールボックス読取・共有カレンダーイベント管理・共有メールボックスからの送信
- **Codex Pro tier**: $100/月、Plus/Proプラン向け長時間セッション対応
- **omni-moderation-latest**: 画像+テキスト対応の新モデレーションモデル、2つの新テキストカテゴリ追加
- https://help.openai.com/en/articles/6825453-chatgpt-release-notes

### Gemini Robotics-ER 1.6（4/14）

### Gemini in Classroom — 全Education エディション対応
