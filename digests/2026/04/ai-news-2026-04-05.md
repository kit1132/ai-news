## Claude Code / Anthropic

### Claude Code v2.1.92 リリース（4/4）
- Bedrock対話型セットアップウィザード追加（AWS認証・クレデンシャル検証付き）
- `/cost` にモデル別・キャッシュヒット別の内訳表示（サブスクリプションユーザー向け）
- Write toolの差分計算速度が大規模ファイルで60%高速化
- `forceRemoteSettingsRefresh` ポリシー設定追加（リモート管理設定のfail-closed取得）
- `/release-notes` がインタラクティブなバージョンピッカーに変更
- Remote Controlセッション名にホスト名をデフォルトプレフィックスとして使用
- `/tag` コマンドと `/vim` コマンドを削除（vimモードは `/config` から切替）
- サブエージェント起動失敗、Stop hookの誤動作、API 400エラー等の複数バグ修正
- Linux sandboxに `apply-seccomp` ヘルパーをnpm/ネイティブ両ビルドで同梱
- https://code.claude.com/docs/en/changelog

### Anthropic マルチエージェントハーネス設計を公開（4/4）
- 長時間自律アプリケーション開発向けの3エージェント構成（Planner / Generator / Evaluator）
- フロントエンドデザインとフルスタック開発がターゲット
- 1回のランで5〜15イテレーション、最大4時間の自律セッション
- 単独実行: 20分/$9 → フルハーネス: 6時間/$200 でより完成度の高い成果物
- https://www.anthropic.com/engineering/harness-design-long-running-apps

### Anthropic、OpenClaw等サードパーティツールをサブスクリプション対象外に（4/4）
- Pro/Maxプランの定額利用枠からOpenClaw等のサードパーティフレームワーク経由の利用を除外
- 継続利用には「extra usage」課金が必要（事前購入で最大30%割引）
- 影響: 推定135,000以上のOpenClawインスタンスが稼働中、最大50倍のコスト増となるユーザーも
- 月額プラン相当の1回限りクレジット付与（4/17まで）
- https://thenextweb.com/news/anthropic-openclaw-claude-subscription-ban-cost
