## 最重要

### OpenAIモデル・Codex・Managed AgentsがAWS上で提供開始（4/28）
GPT-5.5を含むOpenAIモデルがAmazon Bedrockで利用可能に（Limited Preview）。Codex CLI/デスクトップ/VS CodeもBedrockで動作。Bedrock Managed AgentsでOpenAI製エージェントをAWS環境にデプロイ可能。IAM・PrivateLink・Guardrails・CloudTrailによるエンタープライズ統制。Microsoftとの排他契約解消後、初の本格マルチクラウド展開。
- https://openai.com/index/openai-on-aws/
- https://aws.amazon.com/about-aws/whats-new/2026/04/bedrock-openai-models-codex-managed-agents/

### Anthropic-Amazon パートナーシップ拡大（4/20-21）
Amazon追加投資 最大$25B（既存$8B含め合計$33B）。Anthropicは今後10年間で$100B以上のAWSテクノロジー（Trainium含む）利用をコミット。最大5GWの計算容量確保。年内にTrainium2/3で約1GW稼働予定。Anthropic収益は年間ランレート$30B超（2025年末の$9Bから急成長）。
- https://www.anthropic.com/news/anthropic-amazon-compute
- https://techcrunch.com/2026/04/20/anthropic-takes-5b-from-amazon-and-pledges-100b-in-cloud-spending-in-return/

## Claude Code

### v2.1.120〜v2.1.123（4/28〜4/29）— 4リリース
主要な変更:
- **Windows: Git Bash不要に（v2.1.120）** — Git for Windows未インストール時はPowerShellをシェルツールとして使用
- **`claude ultrareview` CLIサブコマンド（v2.1.120）** — CI/スクリプトから非対話的に実行。`--json`で生出力、終了コード0/1
- **MCP `alwaysLoad`（v2.1.121）** — `true`設定でtool-search遅延をスキップし常時利用可能に
- **`/skills`検索ボックス（v2.1.121）** — タイプフィルターでスキルを即座に検索
- **PostToolUseフック拡張（v2.1.121）** — 全ツールでtool出力を`hookSpecificOutput.updatedToolOutput`で置換可能（従来MCPのみ）
- **`claude plugin prune`（v2.1.121）** — 孤立プラグイン依存の自動削除
- **`/resume` PR URL検索（v2.1.122）** — PR URLを貼り付けると該当セッションを検索（GitHub/GitLab/Bitbucket対応）
- **`ANTHROPIC_BEDROCK_SERVICE_TIER`（v2.1.122）** — Bedrock `default`/`flex`/`priority` ティア選択
- **Vertex AI mTLS対応（v2.1.121）** — X.509証明書ベースWorkload Identity Federation
- **メモリリーク修正（v2.1.121）** — 多画像セッションでの数GB RSS膨張、`/usage`の最大2GBリーク修正
- **OAuth 401ループ修正（v2.1.123）** — `CLAUDE_CODE_DISABLE_EXPERIMENTAL_BETAS=1`設定時の認証失敗を修正
